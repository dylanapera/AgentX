# Basic AI Application Deployment
Welcome to this repository!
If you’re new to chat bot building and want to get started quickly, this learning path will help you deploy a simple AI agent on Azure. By the end, you’ll have a working chat bot and a solid foundation for building more advanced solutions.

## What You’ll Learn

- How to set up your environment for AI agent development.
- Deploying an AI-powered application using Azure AI Foundry.
- Hosting your agent on Azure App Services for scalability and reliability.


## Technologies Used

- Azure AI Foundry – Build and manage AI agents.
- Azure App Services – Host applications securely and at scale.


## Pre-requisites
Make sure you have the following installed:

- Python
- Azure CLI
- Azure AI Foundry SDK (with GPT-4o enabled)
